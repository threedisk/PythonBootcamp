### Yang dipelajari
* For Loops

```python
fruits = ["Apple","Peach","Pear"]
for fruit in fruits:
    print(fruit)
    print(fruit+ " Pie")
```
* Range 
```python
for number in range(a,b):
  print(number)
```
dimana a merupakan nilai awal dari range dan b merupakan nilai akhir dari range
```python
for number in range(1,10):
  print(number)
```
Hasilnya adalah `1,2,3,4,5,6,7,8,9`
